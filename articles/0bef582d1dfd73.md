# root "posts#indexとは
```root "posts#index"```は最初に遷移される画面を指定できる。

config/routes.rb
```
Rails.application.routes.draw do

  get "up" => "rails/health#show", as: :rails_health_check

  resources :rooms do
    resources :comments, only: [:create, :index]
  end
  # Defines the root path route ("/")
  root "posts#index"
end
```
