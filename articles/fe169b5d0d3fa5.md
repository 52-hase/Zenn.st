# 開発環境とファイル構成
- `Laravel`の開発環境は`Docker`で構築します。
```:開発環境
・フレームワーク：Laravel Framework 9.x.x(LaravelMix)
・データベース：MYSQL 5.7.36
・DB管理ツール : phpMyAdmin latest
・PHP：PHP 8.0.x
・Nginx：Nginx latest
・Node.js：node 14.18-alpine
・mail：mailhog latest
```
```:ファイル構成
Laravel_Task_app（ルートディレクトリ）※ 任意の名前
├── docker-compose.yml
├── docker 
│   ├── php 
│   │   ├── Dockerfile 
│   │   └── php.ini 
│   └── nginx 
│       └── default.conf 
├── phpMyAdmin
└── src 
    └──  LaravelProject（Laravelのプロジェクトディレクトリ）※ 任意の名前
```
