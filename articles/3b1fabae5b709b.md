# DOMとは？
- Document Object Model の略称です。

- DOM は JavaScript 言語の一部ではなく、ウェブサイトを構築するために使用される Web API の 1 つです。

::::details 

'''
<!DOCTYPE html>
<html>
<head>
    <title>サンプルページ</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>これはサンプルページです。</p>
</body>
</html>


'''
