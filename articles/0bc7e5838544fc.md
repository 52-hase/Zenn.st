# チャットをする画面の作成
ターミナルでコマンド実行
```
$ rails g controller Rooms show
```

```routes.rbファイル```を以下のように修正
```
Rails.application.routes.draw do
  root to: 'rooms#show'
end
```

メッセージを保存するモデルを作る
```
rails g model Message content:string
```
