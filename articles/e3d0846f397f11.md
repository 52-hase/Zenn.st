# 変数
・変数は他のプロクラミング言語と同様に、箱につけられたラベルとイメージするとわかりやすい
1. 先頭に「$」を付ける
2. $の次には任意の文字かアンダースコア(_)にする
3. 変数名の2文字目以降には数値を利用しても良い
4. 大文字小文字は区別されるので注意する
5. camelCase記法(*)



# 変数展開
