# vim とは？
```vim```とは、テキストエディタでファイルの作成・編集・削除を行うことができる。

# vim のモード
| モード        | 機能                  |
| ------------------------- | ---------------------------- |
| ノーモルモード   | カーソル移動、コピー、貼り付け、削除  |
| 挿入モード | 入力、貼り付け、入力補完、単語削除  |
| ビジュアルモード | 範囲指定、コピー、削除、フィルター |
| コマンドラインモード | 保存、終了、文字列の置換 |
