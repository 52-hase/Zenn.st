# 🌀 Tailwind CSS まとめ

## 1. Tailwind CSSとは

- ユーティリティクラスベースのCSSフレームワーク  
- CSSを直接書かず、HTML上でクラス名を組み合わせてデザインを作成  
- コンポーネント設計やレスポンシブ対応が簡単  
- 他のCSSフレームワーク（Bootstrapなど）に比べて柔軟性が高い
