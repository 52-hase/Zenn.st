# Laravelの開発環境をDockerで構築する
・Laravel開発環境をするディレクトリを作成する ⇨ 作成したファイルに移動する
```
$ mkdir Laravel_app

$ cd Laravel_app
```
・作成した`Laravel_app`ディレクトリに更にディレクトリとファイルを追加する
